static-serve usage:
```bash
# Start server for a directory
node ${CLAUDE_PLUGIN_ROOT}/scripts/appdev-cli.mjs static-serve --dir dist
node ${CLAUDE_PLUGIN_ROOT}/scripts/appdev-cli.mjs static-serve --dir dist --spa true

# Stop all servers
node ${CLAUDE_PLUGIN_ROOT}/scripts/appdev-cli.mjs static-serve --stop
```
Returns JSON: { dir, pid, port, spa } on start, { stopped: true } on stop

   ```

Four branches:

| Prompt? | State? | Action |
|---------|--------|--------|
| No | Yes | Auto-resume: read state, call resume-check, show context, jump to action |
| Yes | Yes | Ask: resume existing or start fresh with new prompt |
| Yes | No | Start fresh: proceed to Step 0.5 |
| No | No | Error: nothing to resume, no prompt given |

**Auto-resume flow (No prompt + State exists):**
```
Bash(node ${CLAUDE_PLUGIN_ROOT}/scripts/appdev-cli.mjs get)
```
Show: original prompt, current round, current step.
```
Bash(node ${CLAUDE_PLUGIN_ROOT}/scripts/appdev-cli.mjs resume-check)

taken). Then jump directly to the action returned by resume-check:

- `plan` -> Step 1
