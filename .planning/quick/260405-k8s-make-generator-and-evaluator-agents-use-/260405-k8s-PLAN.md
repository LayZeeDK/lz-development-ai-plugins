<objective>
Enforce msedge (or chrome) browser channel for all Playwright usage across
Generator and Evaluator agents so that Browser Built-in AI APIs (LanguageModel
Prompt API, Summarizer, etc.) are available during tests and evaluation.

Purpose: Bundled Chromium does not support Chrome/Edge Built-in AI APIs. Without
a branded channel, all AI features degrade to fallback code during e2e tests and
critic evaluation, producing false negatives in functionality scoring.

Output: Updated agent instructions and shared references that enforce
`--browser msedge` for playwright-cli and `channel: 'msedge'` for Playwright
Test configs.
</objective>
